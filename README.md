## 🚀 Projects

### 🟣 MiniShell &nbsp;—&nbsp; `C / Linux System Programming`
> A Unix-like shell built entirely in C from scratch.

- Used `fork()`, `exec()`, `wait()`, `dup2()`, and `pipe()` for full process lifecycle management
- Supports piping, I/O redirection, and background process execution
- Deep hands-on understanding of OS internals and Linux system calls

---

### 🟢 TFTP Client-Server &nbsp;—&nbsp; `C / Network Programming`
> UDP-based file transfer protocol implementation from scratch.

- Packet transmission, ACK handling, and error control over unreliable transport
- Reliable communication achieved through custom retransmission logic

---

### 🟡 CAN Based Automation System &nbsp;—&nbsp; `Embedded C / CAN Protocol`
> Real-time embedded node communication over CAN bus.

- Implemented CAN message transmission and reception using Embedded C and UART interfaces
- Verified communication reliability between nodes under varying operating conditions

---

### 🔴 Automated Solar Panel Cleaning Robot &nbsp;—&nbsp; `Embedded Systems / IoT`
> IoT prototype for fully automated solar panel maintenance.

- Dust detection sensors + microcontroller logic + water pump control
- Reduced manual maintenance through continuous automated monitoring
- Improved solar panel efficiency through automated cleaning cycles

---

### 🔵 Inverted Search Engine &nbsp;—&nbsp; `C / Data Structures`
> Multi-file word indexer in C using advanced data structures.

- Linked lists and file handling for indexing words across multiple documents
- Hashing and searching techniques for fast, efficient data lookup
